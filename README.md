# Pokémon Grey

A Pokémon Red mod that replaces Kanto Pokémon with Unova ones, updating certain events, dialogs, trainers, etc. accordingly.

This project is forked from the dissasembled [**Pokémon Red and Blue**][pokered] by pret.
